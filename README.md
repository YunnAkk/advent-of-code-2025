# Advent Of Code 2025
Advent of Code is a series of programming challenges released each December, themed around Christmas, with one two part challenge per day. This repository documents my thought process and problem-solving approach for each day and it also helps me become more proficient with Rust.

## AI and Tool Assistance Disclosure
This project is my own work. I independently develop the ideas, reason through the problems, write the initial drafts and design and implement the code.

I use AI tools in a supporting role, similar to a mentor/senior-developer for things such as:

- improving grammar, clarity, structure and the tone of written text.
- reviewing code and offering possible improvements.
- discussing alternative approaches after I have first produced an initial solution.

AI is not used as a substitute for my own reasoning or learning. I review, test and take responsibility for all material included in this repository.

I also use standard development and research tools where appropriate including rust documentation, search engines, version control tools and third party libraries.
